Conectar Base de Datos en la Nube

Para sincronizar datos entre todos los dispositivos, conecta tu proyecto Firebase gratuito. Se configura una sola vez.

PASOS (5 minutos):
  1. Ve a console.firebase.google.com
  2. Crea un proyecto nuevo
  3. Realtime Database → Crear base de datos → Modo prueba
  4. Configuracion del proyecto → Tus apps → Web → Registrar
  5. Copia los valores de firebaseConfig aqui abajo

    ✉ Configurar EmailJS (Correos Automaticos)

    EmailJS envia correos recordatorio a los colaboradores directamente desde el navegador, sin servidor. Cuenta gratuita: emailjs.com (200 correos/mes gratis)

    Variables de plantilla EmailJS a usar:
    {{to_name}} — Nombre del colaborador
    {{to_email}} — Email del colaborador
    {{pending_modules}} — Modulos pendientes
    {{days_remaining}} — Dias restantes del mes
    {{month_name}} — Nombre del mes actual
